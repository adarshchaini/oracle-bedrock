Issue Tracking

The Issue Tracking system for Oracle Tools is hosted by java.net.

It can be found here: http://java.net/jira/browse/ORACLETOOLS

Raising Issues:

To raise an issue representing a defect or feature callable in the issue tracking system you need to have a java.net account. These are free. They don’t require any personal information beyond that of an email address to confirm registration.

Modifying and Closing Issues:

To modify and close issues with the issue tracking system you will need to be a contributor.
